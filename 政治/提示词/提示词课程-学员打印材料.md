# 【正面】弱vs强提示词对比示例

## 场景：课堂提问设计（价值规律）

### ❌ 弱提示词

```
请为"价值规律"这个知识点设计课堂提问。
```

**AI生成结果（示例）：**
1. 什么是价值规律？
2. 价值规律的内容是什么？
3. 价值规律有什么作用？
4. 请举例说明价值规律在生活中的体现。

**问题分析：**
- ❌ 太简单：都是记忆层面的提问
- ❌ 没有递进：问题之间没有逻辑关系
- ❌ 不结合实际：没有考虑学生的真实情况
- ❌ 缺少引导：老师还要自己想怎么引导

---

### ✅ 强提示词

```
【任务】
请为"价值规律"这个知识点设计层层递进的课堂提问，引导学生从生活现象理解抽象原理。

【背景】
- 知识点：价值规律
- 年级：高二
- 课时：45分钟（第二课时：理解与应用）
- 学生情况：
  * 第一课时已学习价值规律的基本概念（能背定义）
  * 但对概念的理解比较机械，不会应用
  * 不理解为什么"社会必要劳动时间决定价值"
- 教学目标：
  * 通过生活案例理解价值规律
  * 学会用价值规律分析现实经济现象

【要求】
1. 提问层次（4-5层，层层递进）：
   - 第1层：从生活现象引入，激发兴趣
   - 第2层：引导观察和思考，发现规律
   - 第3层：连接原理，理解概念
   - 第4层：应用原理，分析新现象

2. 每个问题要求：
   - 问题表述清晰、具体
   - 给出预期答案（学生可能怎么回答）
   - 给出追问方向（如果学生答偏了，如何引导）

3. 案例选择：
   - 使用高中生熟悉的生活场景
   - 案例要真实、典型

4. 价值观导向：
   - 基于社会主义市场经济理论
   - 体现马克思主义政治经济学原理
```

**AI生成结果（节选）：**

```
## 第1层：从生活现象引入

**问题1：** "同学们，学校门口几家奶茶店，价格都差不多，都是12-15元一杯。为什么不是8元，也不是20元？"

【预期答案】因为成本差不多、市场竞争等
【追问方向】"成本差不多，是什么决定了成本？制作一杯奶茶需要的时间、原料、人工..."

---

## 第2层：引导观察和思考

**问题2：** "10年前，一部智能手机3000-5000元，现在同样配置只要1000-2000元。为什么手机越来越便宜？"

【预期答案】技术进步、生产效率提高
【追问方向】"技术进步，意味着生产一部手机需要的时间减少了..."

---

## 第3层：连接原理

**问题3：** "从这两个例子，我们发现了什么规律？商品的价格，是不是和生产它需要的时间、精力有关？"

【预期答案】是的，花的时间多，价格就高
【追问方向】"这就是价值规律：商品的价值量由社会必要劳动时间决定..."
```

**效果对比：**
- ✅ 层层递进：从现象→规律→原理→应用
- ✅ 有具体指导：每个问题都有预期答案和追问方向
- ✅ 结合实际：考虑了学生的真实情况和教学目标
- ✅ 可直接使用：老师拿到就能用，或稍微调整

---

## 🔑 核心方法：三要素

### 【任务】你要AI做什么事
- ❌ 模糊：设计课堂提问
- ✅ 清晰：设计层层递进的课堂提问，引导学生从生活现象理解抽象原理

### 【背景】在什么情况下做这件事
- 年级、课时、学生情况、教学目标
- 背景越丰富，AI的结果越贴合实际

### 【要求】做到什么程度、什么标准
- 提问层次、每个问题的要求、案例选择、价值观导向
- 要求越细致，AI的结果越符合标准

---

**记住：信息越清楚，结果越好！**

## 场景1：教案生成

### 适用场景
- 你要备一节新课，需要完整的教学设计
- 你想参考不同的教学思路
- 你想快速生成教案框架，再自己完善

### 填空模板

```
【任务】
请生成一份教案，帮助学生理解___________（知识点），侧重___________（概念讲解/案例分析/思维训练）。

【背景】
- 知识点：___________
- 年级：___________
- 课时：___________分钟（第_____课时）
- 学生情况：
  * ___________（如：第一次接触这个概念/已经学过基本内容）
  * ___________（如：容易混淆XX和XX/对抽象概念理解困难）
- 教学目标：
  * ___________（如：理解概念的内涵和外延）
  * ___________（如：学会用概念分析现实问题）

【要求】
1. 教学流程：
   - 导入环节（___________分钟）：___________（如：用时事案例引入）
   - 讲授环节（___________分钟）：___________（如：讲解概念，结合案例）
   - 练习环节（___________分钟）：___________（如：课堂讨论/小组练习）
   - 总结环节（___________分钟）：___________（如：知识点梳理，答疑）

2. 教学方法：
   - ___________（如：案例教学法/问题引导法/对比分析法）

3. 教学资源：
   - 需要___________（如：时事材料/图表数据/视频片段）

4. 价值观导向：
   - 基于中国特色社会主义理论体系
   - 符合高中政治教材的官方表述

5. 输出格式：
   - 结构清晰，分【教学目标】【教学重难点】【教学流程】【板书设计】等模块
   - 适合打印使用
```

## 场景2：概念辨析设计

### 适用场景
- 学生容易混淆两个相似概念
- 你想设计一个对比讲解的教学环节
- 你想帮学生理清概念的区别和联系

### 填空模板

```
【任务】
请设计一个教学环节，帮助学生辨析___________和___________这两个概念的区别与联系。

【背景】
- 两个概念：___________、___________
- 年级：___________
- 学生困惑：
  * ___________（如：不理解两个概念的本质区别）
  * ___________（如：答题时经常混用）
- 教学目标：
  * 理解两个概念的内涵
  * 掌握区分的关键点
  * 能在答题时正确使用

【要求】
1. 对比分析（用表格）：
   - 从___________、___________、___________等角度对比
   - 说明相同点和不同点
   - 指出区分的关键

2. 案例分析：
   - 提供___________个案例
   - 每个案例说明用哪个概念合适，为什么

3. 答题应用：
   - 这两个概念在答题时常见的考查方式
   - 如何判断题目考的是哪个概念
   - 答题时要注意什么

4. 易错提示：
   - 学生容易在哪里混淆
   - 如何避免混淆

5. 价值观导向：
   - 基于马克思主义理论
   - 符合高中政治教材的官方表述
```

## 场景3：材料分析题讲评

### 适用场景
- 考试后要讲评一道材料分析题
- 你想让学生理解答题思路
- 你想强调易错点和答题技巧

### 填空模板

```
【任务】
请为这道材料分析题设计讲评方案，帮助学生理解答题思路和易错点。

【背景】
- 题目类型：___________（如：经济生活材料分析题）
- 考查知识点：___________
- 年级：___________
- 学生答题情况：
  * 平均分：_____分（满分_____分）
  * 主要问题：___________（如：只答原理，不结合材料）

【题目】
（粘贴完整题目和材料）

【标准答案】
（粘贴标准答案）

【要求】
1. 审题指导：
   - 这道题考查什么知识点（如何从题目和材料中判断）
   - 题目要求是什么（如何提取关键信息）
   - 材料的核心信息是什么

2. 答题思路：
   - 标准答案的每个要点为什么要答
   - 如何从材料中提取信息
   - 如何把原理和材料结合起来

3. 易错点分析：
   - 学生容易在哪里答偏（用具体例子说明）
   - 为什么会答偏（思维误区是什么）
   - 如何避免

4. 答题技巧：
   - 这类题目的通用思路是什么
   - 答题结构如何组织
   - 关键词如何使用

5. 举一反三：
   - 类似题目的答题方法
   - 同类知识点的其他考查角度
```

## 场景4：时事热点结合知识点

### 适用场景

- 你想提高学生的学习兴趣

### 填空模板

```
【任务】

【背景】
- 知识点：___________
- 时事热点：___________（简要描述）

  * 通过时事理解知识点
  * 学会用知识点分析现实问题
  * 提高学生学习兴趣

【要求】
1. 时事呈现：
   - 简要介绍时事背景和核心内容
```

## 场景5：学生答疑解惑

### 适用场景
- 学生问"老师，XX概念我不理解"
- 学生问"XX和XX有什么区别"
- 你想生成一个通俗易懂的解释

### 填空模板

```
【任务】
请帮助学生理解___________（概念/问题），用通俗的语言和生活案例解释。

【背景】
- 学生的问题：___________（如："老师，什么是社会必要劳动时间？"）
- 年级：___________
- 学生困惑点：
  * ___________（如：能背定义，但不理解什么是"社会平均"）
  * ___________（如：不知道这个概念有什么用）

【要求】
1. 通俗解释（用大白话）：
   - 这个概念说的是什么事
   - 为什么会有这个概念
   - 核心就是什么（一句话概括）

2. 生活案例（___________个）：
   - 从高中生熟悉的生活场景举例
   - 说明概念在生活中的体现
   - 帮助学生建立直觉理解

3. 答题应用：
   - 这个概念在答题时怎么用
   - 常见的考查角度是什么
   - 答题时要注意什么

4. 常见误区：
   - 学生容易把这个概念和什么混淆
   - 常见的错误理解是什么
   - 如何区分

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 语言要通俗，高中生能看懂
```

## 💡 使用技巧

### 1. 保存你的填写结果
- 第一次填完后，保存下来
- 下次只改关键词就行（如换个知识点、换个年级）

### 2. 多次使用同一个模板
- 这些模板是通用的
- 你只需要改背景信息和具体内容
- 一个模板可以用一整个学期

### 3. AI不满意就追问
- AI的第一次回答不一定完美
- 可以说："能更具体吗？""能举例说明吗？"
- 多问几次，找到最好的答案

### 4. 人工复核很重要
- AI也会出错
- 涉及政治立场、价值观的内容，一定要自己看一遍
- 你是把关人，AI是助手
