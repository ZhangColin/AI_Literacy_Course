# 文生文课程 - 行动卡（政治科）

> **使用说明：** 本行动卡包含5个场景的填空模板，根据你的实际情况填写空白处，然后复制整个提示词到AI工具中使用

---

## 📋 5个核心场景

1. **概念通俗解释** - 帮学生理解抽象概念
2. **答题示范+思路分析** - 讲评材料分析题
3. **学生答案批改与诊断** - 快速批改，给出反馈
4. **案例库生成** - 快速生成生活化案例
5. **时事热点分析** - 用时事结合知识点

## 场景1：概念通俗解释

### 适用场景
- 学生对某个概念不理解（能背定义，但不会用）
- 你想用通俗语言和生活案例帮学生理解

### 填空模板

```
【任务】
请帮助高中生理解"___________"这个概念，用通俗的语言和生活案例解释。

【背景】
- 概念：___________
- 年级：___________
- 模块：___________（经济生活/政治生活/文化生活/生活与哲学）
- 学生困惑：___________（如：能背定义，但不理解；不知道怎么用；和XX概念混淆）

【要求】
1. 通俗解释（用大白话）：
   - 这个概念说的是什么事
   - 为什么会有这个概念
   - 核心就是什么（一句话概括）

2. 生活案例（3个）：
   - 从高中生熟悉的生活场景举例
   - 说明概念在生活中的体现
   - 帮助学生建立直觉理解

3. 答题应用：
   - 这个概念在答题时怎么用
   - 常见的考查角度是什么
   - 答题时要注意什么

4. 常见误区：
   - 学生容易把这个概念和什么混淆
   - 常见的错误理解是什么

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 适合学生自己阅读理解

【价值观导向】
- 基于中国特色社会主义理论体系
- 符合社会主义核心价值观
- 遵循教材的官方表述和主流解读
- 涉及时事时，参考人民日报、新华社等官方媒体观点
```

## 场景2：答题示范+思路分析

### 适用场景
- 你有一道典型的材料分析题，想给学生讲解答题思路
- 学生知道知识点，但不会答题

### 填空模板

```
【任务】
请为这道高中政治材料分析题生成一份___________分水平的答案示例，并详细展示答题思路。

【背景】
- 题目类型：___________（材料分析题/辨析题/探究题等）
- 年级：___________
- 模块：___________（经济生活/政治生活/文化生活/生活与哲学）
- 学生困惑：___________（如：不知道如何结合材料；只写原理，不分析）

【题目】
（请完整粘贴题目和材料）
___________

【标准答案】（如果有）
___________

【要求】
1. 完整答案（___________分水平）：
   - 原理部分：相关原理的准确表述
   - 材料分析部分：如何结合材料说明原理
   - 结论部分：简要总结（如果需要）

2. 思路展示（这是关键）：
   对答案的每个部分，说明：
   - 为什么要答这个点（考查意图）
   - 从材料的哪里提取的信息（材料与原理的结合点）
   - 怎么把原理和材料连接起来（分析过程）
   - 答题的逻辑结构

3. 易错提示：
   - 学生容易在哪里答偏或遗漏
   - 常见的错误理解是什么

4. 答题技巧：
   - 这类题目的通用审题和答题思路是什么
   - 如何组织语言，使答案条理清晰

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 适合老师讲解和学生学习

【价值观导向】
- 基于中国特色社会主义理论体系
- 符合社会主义核心价值观
- 遵循教材的官方表述和主流解读
- 涉及时事时，参考人民日报、新华社等官方媒体观点
```

## 场景3：学生答案批改与诊断

### 适用场景
- 你有学生的答案，想快速诊断问题所在
- 你想给学生个性化的反馈和改进建议

### 填空模板

```
【任务】
请批改这份学生答案，诊断问题所在，并给出具体的改进建议。

【背景】
- 题目类型：___________
- 年级：___________
- 模块：___________
- 满分：___________分
- 学生情况：___________（如：优秀/中等/基础薄弱）

【题目】
（请完整粘贴题目和材料）
___________

【标准答案】
___________

【学生答案】
（请完整粘贴学生的实际答案）
___________

【要求】
1. 评分（给出具体分数）：
   - 这份答案能得多少分（满分___________分）
   - 每个要点的得分情况

2. 问题诊断（详细分析）：
   - 答对了什么（肯定优点）
   - 答错了什么或遗漏了什么（指出问题）
   - 为什么会这样答（分析思维误区）

3. 改进建议（可操作）：
   - 这份答案如何改进才能得高分
   - 学生需要注意什么（如表述规范、结合材料等）
   - 下次答题时的注意事项

4. 个性化反馈：
   - 针对这个学生的水平，给出鼓励和建议
   - 语气要友善，但要明确指出问题

5. 输出格式：
   - 分【评分】【问题诊断】【改进建议】【给学生的话】模块
   - 适合直接反馈给学生

【价值观导向】
- 基于教材的官方表述和主流解读
- 确保批改的价值观正确性
```

## 场景4：案例库生成

### 适用场景
- 你要讲某个知识点，需要多个生活化案例
- 教材上的案例太老旧，想找更贴近学生的

### 填空模板

```
【任务】
请为"___________"这个知识点生成___________个生活化案例，帮助高中生理解。

【背景】
- 知识点：___________
- 年级：___________
- 模块：___________
- 教学目的：___________（如：通过生活案例帮助学生理解抽象概念）

【要求】
1. 案例要求：
   - 从高中生熟悉的生活场景选取（如___________）
   - 案例要真实、典型、容易引起共鸣
   - 案例要能清楚说明___________（知识点的核心）

2. 每个案例包含：
   - 案例描述（简洁，100-150字）
   - 知识点分析（如何体现这个知识点）
   - 启示或结论（这个案例告诉我们什么）

3. 案例的多样性：
   - ___________个案例要从不同角度切入
   - 覆盖学生生活的不同方面
   - 有简单的、有复杂的，层次丰富

4. 价值观导向：
   - 案例要积极向上，符合社会主义核心价值观
   - 避免敏感话题
   - 体现正确的人生观、价值观

5. 输出格式：
   - 分【案例1】【案例2】...【案例N】展示
   - 每个案例结构清晰，有小标题
   - 适合老师课堂讲解或学生自己阅读
```

## 场景5：时事热点分析

### 适用场景
- 你想用时事热点引入课堂，提高学生兴趣
- 你想让学生学会用政治知识分析现实问题

### ⚠️ 特别提醒
**时事分析的价值观要求非常严格，使用前必须：**
1. 参考人民日报、新华社等官方媒体的权威解读
2. 使用国内AI（DeepSeek、豆包、Kimi）
3. AI生成后，必须人工审核
4. 不确定的地方，查教材或问同事

### 填空模板

```
【任务】
请用"___________"这个时事热点，结合"___________"知识点，设计一份教学材料。

【背景】
- 知识点：___________
- 时事热点：___________（简要描述）
- 年级：___________
- 模块：___________
- 教学目的：
  * ___________（如：通过时事理解知识点）
  * ___________（如：培养学生分析现实问题的能力）

【要求】
1. 时事背景：
   - 简要介绍时事的背景和核心内容（200-300字）
   - 提炼关键信息（如政策要点、数据、措施等）

2. 知识点关联：
   - 这个时事可以用哪些知识点分析（列出相关原理）
   - 为什么用这些知识点（时事与原理的逻辑关系）

3. 深度分析：
   - 用知识点分析时事（原理+时事材料分析）
   - 体现了什么原理、为什么会这样、有什么意义
   - 分析要深入，不能浮于表面

4. 课堂讨论问题设计（3-5个）：
   - 从不同角度引导学生思考
   - 问题要有层次（从简单到复杂）
   - 帮助学生深化理解

5. 答题应用：
   - 如果考试以这个时事为背景出题，可能怎么考
   - 学生应该怎么答（给出答题思路）

6. 价值观导向（非常重要）：
   - 必须基于中国特色社会主义理论体系
   - 严格符合社会主义核心价值观
   - 参考人民日报、新华社等官方媒体的权威解读
   - 体现党的领导和制度优势
   - 遵循教材的官方表述和主流解读

7. 输出格式：
   - 分【时事背景】【知识关联】【深度分析】【讨论问题】【答题应用】模块
   - 结构清晰，适合课堂使用
```

## 💡 使用技巧

### 技巧1：保存你的填写结果
- 第一次填完后，保存到手机备忘录或电脑文档
- 下次只改关键词（如知识点、年级、题目）
- 不用每次都重新填

### 技巧2：建立自己的模板库
```
我的常用模板：
- 价值规律_概念解释.txt（已填好价值规律的信息）
- 矛盾观_概念解释.txt（已填好矛盾观的信息）
- ...
```

### 技巧3：追问优化结果
- AI的第一次回答不满意？直接追问：
  - "能更详细吗？"
  - "案例能换一个吗？"
  - "能用更通俗的语言吗？"

### 技巧4：批量使用
- 批改多份学生答案？可以一次批改一份，保存好提示词
- 需要多个案例？可以一次生成8-10个，再挑选
