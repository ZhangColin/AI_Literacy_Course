# 测试：古诗文教学辅助

## 提示词

【任务】
请为这首古诗设计一份教学辅助材料，帮助高中语文老师备课。

【背景】
- 诗歌：《春望》- 杜甫
- 年级：高一
- 课时：45分钟
- 学生基础：对杜甫有基本了解，古文阅读能力中等
- 教学目标：理解诗歌内容、赏析艺术手法、体会作者情感

【诗歌原文】
国破山河在，城春草木深。
感时花溅泪，恨别鸟惊心。
烽火连三月，家书抵万金。
白头搔更短，浑欲不胜簪。

【要求】
1. 逐句翻译（白话文，准确但不生硬）

2. 整体赏析：
   - 写作背景（简要）
   - 主要内容和主题
   - 情感基调

3. 重点讲解（用于课堂）：
   - 意象分析：选3-4个关键意象，说明作用
   - 艺术手法：选2-3个手法，结合具体诗句讲解
   - 情感层次：诗歌的情感是如何层层推进的

4. 教学建议：
   - 课堂上重点讲哪几个点（按优先级）
   - 哪些地方学生容易理解错
   - 可以设计什么问题引导学生思考

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 适合老师快速阅读和使用
