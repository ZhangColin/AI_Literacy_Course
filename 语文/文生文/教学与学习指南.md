# 文生文课程 - 学习指南（语文科）

> **课程目标：** 掌握现成的提示词，改几个关键词就能用  
> **课程性质：** 实操课，不讲理论，重点是效果和实践

---

## 课程内容

本课程提供5个真实场景的AI提示词，帮助你快速解决语文教学中的常见问题：

| 场景 | 解决什么问题 |
|------|-------------|
| 场景1：作文批改+反馈生成 | 快速批改学生作文，生成结构化反馈 |
| 场景2：好坏作文对比分析 | 用AI分析差距，教学生写好作文 |
| 场景3：古诗文教学辅助 | 翻译、赏析、教学建议一站式生成 |
| 场景4：作文升级建议 | 从75分到85分的具体改进路径 |
| 场景5：个性化评语生成 | 快速生成针对性评语 |

## 场景1：作文批改+反馈生成

### 使用场景
- 你需要批改一大堆作文，时间紧张
- 你想给学生结构化的反馈（不只是分数）
- 你想让AI帮你初步诊断，再人工完善

### 完整提示词

```
【任务】
请批改这篇学生作文，给出结构化的反馈。

【背景】
- 年级：高二
- 作文类型：议论文
- 字数要求：800字
- 评分标准：满分120分
- 学生水平：中等（平时考试75-80分）

【要求】
1. 诊断维度（用表格）：
   - 内容：主题是否明确，内容是否充实
   - 结构：开头、主体、结尾是否完整，逻辑是否清晰
   - 语言：词汇是否丰富，句式是否多样，有无病句
   - 思想：立意是否深刻，是否有思考深度

2. 具体问题（列出3-5个主要问题）：
   - 每个问题标注【问题类型】【具体位置】【严重程度】
   - 举例说明问题所在
   
3. 优点（列出2-3个亮点）：
   - 具体说明哪里写得好，为什么好
   - 引用作文中的具体句子

4. 改进建议（给出3条具体建议）：
   - 按优先级排序
   - 每条说明：改什么、怎么改、为什么
   - 预计每条能提升多少分

5. 预估分数：
   - 给出分数（如78/120）
   - 说明主要扣分点

【作文内容】
（这里粘贴学生作文原文）
```

---

### 使用建议

**修改要点：**
- 根据实际情况修改年级、作文类型、字数要求
- 根据学生平时水平填写"学生水平"（如"优秀90分以上""基础薄弱60分左右"）
- 把学生作文原文粘贴到最后

**注意事项：**
1. **AI的批改要人工复核**
   - 检查有无明显错误
   - 调整语气和表达
   - 加入针对性建议

2. **不要完全依赖AI的分数**
   - AI给的分数是参考
   - 你要根据实际情况调整

3. **批改结果要简化后给学生**
   - AI的反馈可能太专业
   - 挑选学生能理解的部分
   - 用学生能接受的语言表达

## 场景2：好坏作文对比分析

### 使用场景
- 你想教学生"好作文好在哪"
- 你想给学生讲解如何从低分提升到高分
- 你想用具体例子让学生看到差距

### 完整提示词

```
【任务】
请对比分析这两篇学生作文，说明它们的差距在哪里。

【背景】
- 作文A：65分水平（基础薄弱学生）
- 作文B：85分水平（优秀学生）
- 对比目的：用于课堂讲解，让学生看到好作文和普通作文的具体差距
- 题目：挫折是成长的阶梯

【要求】
1. 逐项对比（用表格）：
   - 内容：作文A的问题 vs 作文B的优点（引用具体例子）
   - 结构：作文A的单薄 vs 作文B的完整（具体说明差在哪）
   - 语言：作文A的平淡 vs 作文B的生动（举例5处对比）
   - 思想：作文A的浅显 vs 作文B的深刻（具体分析）
   - 整体：给人的感觉有什么不同（评卷老师视角）

2. 提升路径：
   - 如果作文A想达到作文B的水平，优先改什么（前3条）
   - 每条预计能提升多少分
   - 为什么这3条最重要

3. 教学建议：
   - 如果课堂上讲这个对比，重点讲哪3点最有说服力
   - 每个点用这两篇作文的哪个具体例子
   - 学生最容易理解的讲解顺序

【作文A（65分）】
（这里粘贴低分作文原文）

【作文B（85分）】
（这里粘贴高分作文原文）
```

---

### 推荐的作文组合

**组合1（推荐）：60-65分 vs 85分**
- 差距最明显，对比效果最好
- 适合大部分学生看

**组合2：70-75分 vs 85分**
- 适合讲提升技巧
- 重点在优化而非纠错

**组合3：75-80分 vs 90分**
- 差距较小，适合冲刺班
- 重点讲细节优化和思想深度

---

### 注意事项

1. **两篇作文要同一题目**
   - 否则对比没意义
   - 确保可比性

2. **关注具体例子**
   - AI会给出对比表格
   - 重点看具体例子：低分怎么写，高分怎么写
   - 这些例子最有说服力

3. **提升路径要可操作**
   - AI给出的优先级很重要
   - 告诉学生"改这3条，就能提升XX分"
   - 给学生信心

## 场景3：古诗文教学辅助

### 使用场景
- 你要备课，需要翻译、赏析、知识点讲解
- 你想快速生成教学材料
- 你想给学生准备完整的学习资料

### 完整提示词

```
【任务】
请为这首古诗设计教学辅助材料，帮助语文老师备课。

【背景】
- 诗歌：《春望》- 杜甫
- 年级：高一
- 课时：45分钟
- 学生基础：对杜甫有基本了解，古文阅读能力中等
- 教学目标：理解诗歌内容、赏析艺术手法、体会作者情感

【诗歌原文】
国破山河在，城春草木深。
感时花溅泪，恨别鸟惊心。
烽火连三月，家书抵万金。
白头搔更短，浑欲不胜簪。

【要求】
1. 逐句翻译（白话文，准确但不生硬）

2. 整体赏析：
   - 写作背景（简要）
   - 主要内容和主题
   - 情感基调

3. 重点讲解（用于课堂）：
   - 意象分析：选3-4个关键意象，说明作用
   - 艺术手法：选2-3个手法，结合具体诗句讲解
   - 情感层次：诗歌的情感是如何层层推进的

4. 教学建议：
   - 课堂上重点讲哪几个点（按优先级）
   - 哪些地方学生容易理解错
   - 可以设计什么问题引导学生思考

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 适合老师快速阅读和使用
```

---

### 使用建议

**修改要点：**
- 把诗歌名称、作者换成你要讲的
- 根据年级调整"学生基础"（初中/高中/大学）
- 把古诗原文粘贴进去

**适用古诗文类型：**
- 古诗（五言、七言、绝句、律诗）
- 词（如《水调歌头》《念奴娇·赤壁怀古》）
- 文言文短篇（如《陋室铭》《爱莲说》）

---

### 注意事项

1. **AI的分析要审核**
   - 翻译是否准确
   - 赏析是否到位
   - 有没有明显错误

2. **不要机械照搬**
   - AI的讲解可能比较"百度百科"
   - 你要加入自己的理解和教学经验
   - 让内容更生动、更贴近学生

3. **结合课堂实际**
   - AI给的建议是通用的
   - 你要根据自己班级情况调整

## 场景4：作文升级建议

### 使用场景
- 你想帮学生把作文从中等水平提升到优秀
- 你想给学生具体的改进路径
- 你想让学生知道"怎么改能提高分数"

### 完整提示词

```
【任务】
请为这篇学生作文提供具体的升级建议，帮助学生将作文从75分提升到85分以上。

【背景】
- 当前作文水平：75分（满分120分）
- 目标分数：85分以上
- 年级：高二
- 文体：议论文
- 题目：挫折是成长的阶梯

【要求】
1. 水平诊断：
   - 当前作文的优点（2-3条，引用具体句子）
   - 当前作文的主要问题（3-5条，按影响分数的程度排序）
   - 为什么是75分而不是85分？（核心差距在哪里）

2. 具体升级建议（3-5条，按优先级排序）：
   每条建议必须包括：
   - 改什么：具体到第几段、哪句话
   - 怎么改：给出具体的修改方案或方向
   - 为什么：解释为什么这样改能提升分数
   - 预期提升：这一条大概能提升多少分

3. 示例对比：
   - 选1-2处最关键的地方
   - 展示"改前 vs 改后"
   - 说明差距在哪里

4. 行动清单：
   - 学生可以按什么顺序改
   - 每步预计花多少时间
   - 改完后如何自我检查

【学生作文】
（这里粘贴学生作文原文）
```

---

### 使用示例

**从75分提升到85分的关键方向：**
- 论证从"举例"到"分析"（提升5分）
- 语言从"口语化"到"有文学性"（提升3分）
- 思想从"单一"到"有思辨"（提升2-3分）

---

### 注意事项

1. **目标要现实**
   - 不要一次跨度太大（60分→90分不现实）
   - 一般提升10-15分是合理的
   - 可以分阶段：先75→80，再80→85

2. **建议要具体**
   - 不要说"要更深入"（太模糊）
   - 要说"第2段举完例子后，加一句分析为什么"（具体可操作）

3. **改前改后要对比**
   - 给学生看具体例子
   - 让学生明白"好"好在哪里

## 场景5：个性化评语生成

### 使用场景
- 期末要给50个学生写评语，时间紧张
- 你想给每个学生针对性的评价
- 你想让评语有温度、有鼓励

### 完整提示词

```
【任务】
请为这个学生生成期末评语。

【背景】
- 学生姓名：张同学
- 年级：高二
- 作文水平：语言通顺流畅，但论证深度不够，缺少思辨性思考
- 课堂表现：积极参与讨论，善于提问，对古诗文鉴赏特别有兴趣
- 进步情况：本学期作文从70分提升到78分，进步明显

【要求】
1. 评语结构：
   - 肯定优点（2-3句，结合具体表现）
   - 指出不足（1-2句，委婉表达）
   - 给予期待（1句，鼓励性）

2. 语言风格：
   - 温暖、鼓励，让学生感受到关心
   - 具体，不要空洞的夸奖或批评
   - 适度，不过分夸张也不过于严厉

3. 字数：80-120字

4. 避免：
   - 千篇一律的套话（如"该生表现良好"）
   - 只表扬不指出问题
   - 让学生感到挫败的表达
```

---

### 使用建议

**修改要点：**
- 把学生姓名、年级改成实际的
- "作文水平""课堂表现""进步情况"要具体描述，不要只写"优秀""中等"
- 如果有之前写过满意的评语，可以在要求后面加【示例】，粘贴进去，AI会模仿风格

**个性化技巧：**
- 加入你的观察（如"特别喜欢杜甫的诗""擅长写散文"）
- 加入具体事例（如"上次作文《春天》写得很生动"）
- 用学生的特点（如性格、特长、进步轨迹）

---

### 注意事项

1. **AI生成的评语要修改**
   - AI不了解你的学生
   - 你要加入真实的观察和情感
   - 让评语有温度

2. **不要批量生成后直接用**
   - 每条评语要看一遍
   - 确保符合学生实际
   - 调整语气和表达

3. **保持一致性**
   - 全班评语的风格要相对一致
   - 不要有的很详细，有的很简单

## 常见问题

**问题1："AI生成的反馈太专业，学生看不懂怎么办？"**
> "正常的。AI生成后，你可以追问：'请用高中生能理解的语言重新表达'，或者你自己简化一下再给学生。"

**问题2："我的学生作文和示例不一样，能用吗？"**
> "当然能用。这些提示词是通用的，你把自己学生的作文粘贴进去就行，背景信息根据实际情况改一下。"

**问题3："AI批改的作文有错误怎么办？"**
> "正常的，AI也会出错。你看一遍，有错误就追问AI：'第3段分析有误，请重新分析'。AI是助手，你是把关人。"

**问题4："古诗文的赏析太机械，不够生动？"**
> "可以追问：'能不能更生动一点？用学生容易理解的比喻来解释'，或者你自己加入一些课堂上的生动讲解。"

**问题5："升级建议不够具体，怎么办？"**
> "追问：'第2条建议能详细说说吗？具体到第几段、哪句话、怎么改？'多问几次，AI就会给出更具体的答案。"

**问题6："评语都差不多，没有个性化？"**
> "背景信息要写具体。不要只写'优秀'，要写'语言流畅但论证不够深入'；不要只写'进步了'，要写'从70分提升到78分'。信息越具体，评语越个性化。"

## 补充说明

### 推荐AI工具
- **DeepSeek**：批改作文、对比分析效果好，深度够，适合议论文、古诗文赏析
- **豆包**：对话流畅，生成评语、教学辅助效果不错，语言更自然
- **Kimi**：适合长文本处理（批量批改、长篇古文、多篇对比）

### 课后资源
- 可以准备不同分数段的示例作文（60分、70分、75分、85分、90分）
- 用于场景2（对比分析）和场景4（升级建议）
- 也可以用自己学生的真实作文（效果更好）

### 使用技巧
- **保存提示词**：把这5个提示词保存到手机备忘录或电脑文档，每次只改关键词就行
- **初次使用**：先用一两篇作文测试效果，满意了再批量用
- **不满意就追问**：AI的第一次回答不一定完美，可以说"能更具体吗""能举例说明吗"
- **记录好用的调整**：如果你发现某个提示词加一句话效果更好，记下来，下次直接用
