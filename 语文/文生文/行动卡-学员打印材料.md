# 文生文课程 - 行动卡（语文科）

> **使用说明：** 把这个文档打印出来，课上直接在空白处填写，然后粘贴到AI工具里使用

---

## 场景1：作文批改+反馈生成

### 填空模板

```
【任务】
请批改这篇学生作文,给出结构化的反馈。

【背景】
- 年级：_______
- 作文类型：_______（记叙文/议论文/散文/应用文）
- 字数要求：_______字
- 评分标准：满分_______分
- 学生水平：_______（优秀/中等/基础薄弱）

【要求】
1. 诊断维度（用表格）：
   - 内容：主题是否明确，内容是否充实
   - 结构：开头、主体、结尾是否完整，逻辑是否清晰
   - 语言：词汇是否丰富，句式是否多样，有无语法错误
   - 思想：立意是否深刻，是否有思考深度

2. 具体问题（列出3-5个主要问题）：
   - 每个问题标注【问题类型】【具体位置】【严重程度】
   
3. 优点（列出2-3个亮点）：
   - 具体说明哪里写得好，为什么好

4. 改进建议（给出3条具体建议）：
   - 按优先级排序
   - 每条说明：改什么、怎么改、为什么

5. 预估分数：
   - 给出分数（如75/120）
   - 说明扣分点

【作文内容】
（粘贴学生作文）
```

**快速填写提示：**
- 年级：高一/高二/高三
- 作文类型：记叙文/议论文/散文
- 字数要求：800/1000/1200
- 评分标准：满分120/150
- 学生水平：根据平时表现填

## 场景2：好坏作文对比分析

### 填空模板

```
【任务】
请对比分析这两篇学生作文，说明它们的差距在哪里。

【背景】
- 作文A：_______分水平（基础薄弱/中等）
- 作文B：_______分水平（优秀）
- 对比目的：用于课堂讲解，让学生看到好作文和普通作文的具体差距
- 题目：_______

【要求】
1. 逐项对比（用表格）：
   - 内容：作文A的问题 vs 作文B的优点（具体例子）
   - 结构：作文A的单一 vs 作文B的完整（具体说明）
   - 语言：作文A的简单词 vs 作文B的高级词（举例5处）
   - 思想：作文A的浅显 vs 作文B的深刻（具体分析）
   - 整体：给人的感觉有什么不同

2. 提升路径：
   - 如果作文A想达到作文B的水平，优先改什么（前3条）
   - 每条预计能提升多少分
   - 为什么这3条最重要

3. 教学建议：
   - 如果课堂上讲这个对比，重点讲哪3点最有说服力
   - 每个点用这两篇作文的哪个具体例子
   - 学生最容易理解的讲解顺序

【作文A（低分）】
（粘贴低分作文）

【作文B（高分）】
（粘贴高分作文）
```

**快速填写提示：**
- 推荐组合：60分 vs 85分（效果最好）
- 作文A、B必须是同一题目
- 可以用学生的真实作文

## 场景3：古诗文教学辅助

### 填空模板

```
【任务】
请为这首古诗/古文设计教学辅助材料，帮助语文老师备课。

【背景】
- 作品：_______（作品名+作者）
- 年级：_______
- 课时：_______分钟
- 学生基础：_______（对作者有基本了解吗？古文阅读能力如何？）
- 教学目标：_______（理解内容？赏析手法？体会情感？）

【要求】
1. 逐句翻译（白话文，准确但不生硬）

2. 整体赏析：
   - 写作背景（简要）
   - 主要内容和主题
   - 情感基调

3. 重点讲解（用于课堂）：
   - 意象分析：选3-4个关键意象，说明作用
   - 艺术手法：选2-3个手法，结合具体诗句讲解
   - 情感层次：情感是如何层层推进的

4. 教学建议：
   - 课堂上重点讲哪几个点（按优先级）
   - 哪些地方学生容易理解错
   - 可以设计什么问题引导学生思考

5. 输出格式：
   - 结构清晰，有小标题
   - 重点用【】标注
   - 适合老师快速阅读和使用

【作品原文】
（粘贴古诗/古文原文）
```

**快速填写提示：**
- 作品：《春望》-杜甫 / 《水调歌头》-苏轼
- 年级：高一/高二/高三
- 课时：45分钟（一节课）/ 90分钟（两节课）
- 学生基础：对作者有基本了解，古文阅读能力中等
- 教学目标：理解内容+赏析手法+体会情感

## 场景4：作文升级建议

### 填空模板

```
【任务】
请为这篇学生作文提供具体的升级建议，帮助学生将作文从_____分提升到_____分以上。

【背景】
- 当前作文水平：_____分（满分_____分）
- 目标分数：_____分以上
- 年级：_____
- 文体：_____（记叙文/议论文/散文）
- 题目：_____

【要求】
1. 水平诊断：
   - 当前作文的优点（2-3条）
   - 当前作文的主要问题（3-5条，按影响分数的程度排序）
   - 为什么是XX分而不是XX分？（核心差距在哪里）

2. 具体升级建议（3-5条，按优先级排序）：
   每条建议必须包括：
   - 改什么：具体到第几段、哪句话
   - 怎么改：给出具体的修改方案或方向
   - 为什么：解释为什么这样改能提升分数
   - 预期提升：这一条大概能提升多少分

3. 示例对比：
   - 选1-2处最关键的地方
   - 展示"改前 vs 改后"
   - 说明差距在哪里

4. 行动清单：
   - 学生可以按什么顺序改
   - 每步预计花多少时间
   - 改完后如何自我检查

【学生作文】
（粘贴作文）
```

**快速填写提示：**
- 当前水平：70/75/80分
- 目标分数：提升10-15分最合理（如75→85）
- 不要跨度太大（60→90不现实）
- 年级、文体要填准确

## 场景5：个性化评语生成

### 填空模板

```
【任务】
请为这个学生生成期末评语。

【背景】
- 学生姓名：_____
- 年级：_____
- 作文水平：_____（描述：如"语言通顺但缺乏深度""有思想但表达不够流畅"）
- 课堂表现：_____（如"积极参与，善于提问""比较安静，但认真听讲"）
- 进步情况：_____（如"本学期有明显进步""一直保持稳定""需要加强"）

【要求】
1. 评语结构：
   - 肯定优点（2-3句）
   - 指出不足（1-2句，委婉表达）
   - 给予期待（1句，鼓励性）

2. 语言风格：
   - 温暖、鼓励，让学生感受到关心
   - 具体，不要空洞的夸奖或批评
   - 适度，不过分夸张也不过于严厉

3. 字数：80-120字

4. 避免：
   - 千篇一律的套话
   - 只表扬不指出问题
   - 让学生感到挫败的表达

【示例】
（如果有之前写的满意的评语，可以粘贴作为参考）
```

**快速填写提示：**
- 学生姓名：可以用姓氏代替（如"张同学"）
- 作文水平、课堂表现、进步情况：要具体描述，不要只写"优秀""中等"
- 如果有以前写过满意的评语，粘贴作为示例，AI会模仿风格

## 使用技巧

### 技巧1：保存你的填写结果
- 第一次填完后，拍照或复制保存
- 下次只改关键词就行（如换个题目、换个学生）

### 技巧2：多次使用同一个模板
- 这些模板是通用的
- 你只需要改背景信息和作文内容
- 一个模板可以用一整个学期

### 技巧3：AI不满意就追问
- AI的第一次回答不一定完美
- 可以说："能更具体吗？""能举例说明吗？"
- 多问几次，找到最好的答案

### 技巧4：人工复核很重要
- AI也会出错
- 批改、评语、建议都要人工检查
- 你是把关人，AI是助手
