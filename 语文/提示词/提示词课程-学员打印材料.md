# 【正面】弱提示词 vs 强提示词对比

## 场景：课堂提问设计（为《春望》设计课堂问题）

### ❌ 弱提示词

```
给《春望》设计几个课堂问题
```

**AI输出（效果不好）：**
1. 这首诗的作者是谁？
2. 这首诗写于什么时期？
3. 诗中表达了什么情感？
4. "国破山河在"是什么意思？
5. 诗人为什么说"家书抵万金"？

**问题分析：**
- 只有任务，没有背景和要求
- AI给出的问题太表面，缺乏深度
- 都是记忆性问题，没有思考性
- 不能直接用于课堂

---

### ✅ 强提示词（三要素）

```
【任务】
为《春望》这节课设计课堂提问，引导学生深入理解诗歌。

【背景】
- 课文：杜甫《春望》
- 年级：高一
- 学生基础：对杜甫有基本了解，古文阅读能力中等
- 课时：45分钟
- 教学目标：理解诗歌内容、赏析艺术手法、体会家国情怀

【要求】
1. 问题类型：理解类2个、分析类3个、思辨类1个
2. 难度层次：由浅入深，逐步引导
3. 启发性：不要是非题，要能引发讨论
4. 输出格式：每个问题标注【类型】【目的】【追问】【预期答案要点】
```

**AI输出（效果很好）：**

**【问题1 - 理解类】**
首联"国破山河在，城春草木深"，为什么不说"国破山河破"？"山河在"说明什么？

- 【目的】引导学生关注对比手法
- 【追问】山河在但国破，你觉得诗人心情如何？
- 【预期答案】山河依旧，但国家破碎；对比手法突出人事变迁的悲痛

**【问题2 - 分析类】**
颔联"感时花溅泪，恨别鸟惊心"，花和鸟真的会溅泪、惊心吗？诗人为什么这样写？

- 【目的】理解拟人手法和移情于物
- 【追问】如果直接说"我看到花就落泪"，效果一样吗？
- 【预期答案】拟人手法；移情于物，让情感表达更含蓄深远

（共6个问题，由浅入深，结构完整，可直接用于课堂）

---

## 对比总结：三要素法的威力

| 维度 | 弱提示词 | 强提示词 |
|------|----------|----------|
| 明确任务 | ❌ 模糊 | ✅ 清晰（设计课堂提问，引导深入理解） |
| 丰富背景 | ❌ 没有 | ✅ 年级、学生基础、课时、教学目标 |
| 细化要求 | ❌ 没有 | ✅ 问题类型、难度、启发性、输出格式 |
| **输出质量** | ❌ 表面、不能用 | ✅ 深入、可直接用 |

---

## 三要素法（核心方法）

### 1. 明确任务
**你要AI做什么？**
- 用动词开头：批改、设计、生成、分析
- 说清楚做什么，不要模糊
- 一次只做一件事

### 2. 丰富背景
**提供必要信息**
- 年级和学生基础
- 课时
- 教学目标
- 其他相关信息

### 3. 细化要求
**你希望输出什么样的结果？**
- 输出格式（表格、【】、编号）
- 重点关注什么
- 什么不要做

---

## 高级技巧

### 技巧1：结构化输出
在要求里明确输出格式：
- "请用表格列出..."
- "请用【优点】【问题】【建议】的格式..."
- "请分点列出，每点包括..."

### 技巧2：迭代优化
不满意就追问：
- "能不能更具体一点？"
- "能不能给个例子？"
- "第X条能详细说说吗？"

### 技巧3：角色扮演
让AI站在专家视角：
- "你是一位资深的语文教研专家..."
- "你是一位经验丰富的语文老师..."

## 场景1：课堂提问设计

```
【任务】
为《_______》（课文名）这节课设计课堂提问，引导学生深入理解。

【背景】
- 课文：_______（课文名+作者）
- 年级：_______
- 学生基础：_______（如：古文阅读能力中等、对作者有基本了解）
- 课时：_______分钟
- 教学目标：_______（如：理解诗歌内容、赏析艺术手法、体会情感）

【要求】
1. 问题类型：理解类__个、分析类__个、思辨类__个
2. 难度层次：由浅入深，逐步引导
3. 启发性：不要是非题，要能引发讨论
4. 输出格式：每个问题标注【类型】【目的】【追问】【预期答案要点】
```

**适用场景：** 备课时设计课堂问题

---

## 场景2：教案生成

```
【任务】
为《_______》（课文名）设计一份教案。

【背景】
- 课文：_______（课文名+作者）
- 年级：_______
- 学生基础：_______（描述学生情况）
- 课时：_______节课
- 教学目标：
  1. _______
  2. _______
  3. _______

【要求】
1. 教学环节：导入（__分钟）、新授（__分钟）、练习（__分钟）、小结（__分钟）
2. 教学重点：_______
3. 教学难点：_______
4. 教学方法：_______（如：讨论法、讲授法、合作学习）
5. 板书设计：提供板书框架
6. 课堂活动：设计2-3个学生活动
7. 输出格式：结构清晰，每个环节包括【时间】【内容】【教师活动】【学生活动】
```

**适用场景：** 备课时生成教案框架

---

## 场景3：阅读理解出题

```
【任务】
基于这篇文章，设计阅读理解题目。

【背景】
- 文章：_______（粘贴文章或说明文章名）
- 年级：_______
- 题目用途：_______（练习/测验/考试）
- 考查重点：_______（理解能力/分析能力/鉴赏能力）

【要求】
1. 题目类型：
   - 客观题：___道（选择题、判断题）
   - 主观题：___道（简答题、分析题）
2. 难度分布：简单30%、中等50%、困难20%
3. 考查点：明确每道题考查什么（如：理解文意、分析手法、鉴赏情感）
4. 输出格式：
   - 题目（清晰表述）
   - 参考答案（详细）
   - 评分标准（每题X分，答出X得X分）
```

**适用场景：** 出练习题或考试题

---

## 场景4：作文题目设计

```
【任务】
为单元写作设计作文题目。

【背景】
- 单元主题：_______（如：成长、责任、自然、传统文化）
- 年级：_______
- 写作类型：_______（记叙文/议论文/散文/应用文）
- 训练重点：_______（如：语言表达/结构安排/思想深度/细节描写）

【要求】
1. 题目数量：提供3个不同角度的题目供选择
2. 每个题目包括：
   - 题目本身（有吸引力，不太宽泛也不太狭窄）
   - 写作提示（引导学生思考方向，2-3个角度）
   - 素材建议（可以写什么，举3-5个例子）
   - 注意事项（这个题目容易写偏的地方）
3. 难度适中：既有挑战，又不让学生无从下手
4. 输出格式：清晰结构，便于学生理解
```

**适用场景：** 单元写作教学，设计作文题

---

## 场景5：教学反思撰写

```
【任务】
为这节课撰写教学反思。

【背景】
- 课文：_______
- 年级：_______
- 课堂情况简述：
  _______（如：学生参与度高/低，某个环节用时过长/过短，
  某个知识点学生理解困难，课堂气氛活跃/沉闷等）
- 反思重点：_______（想反思哪方面：教学方法/学生反应/时间把控/教学效果）

【要求】
1. 反思维度：
   - 教学目标达成情况（是否达成？哪些达成了？哪些没达成？）
   - 教学方法的有效性（什么方法有效？什么方法效果不好？）
   - 学生反应和参与度（学生是否积极？哪些环节参与度高/低？）
   - 需要改进的地方（具体问题是什么？）
2. 具体事例：结合课堂实际情况分析（不要空谈，要有具体例子）
3. 改进措施：针对问题提出3-5条具体改进方案（可操作的）
4. 输出格式：
   【课堂亮点】...
   【存在问题】...
   【原因分析】...
   【改进措施】...
   【预期效果】...
```

**适用场景：** 课后撰写教学反思

## 使用建议

### 1. 模板使用步骤
1. 选择合适的场景模板
2. 填写【】中的内容（你的具体信息）
3. 根据需要调整要求（可以增删）
4. 粘贴到AI工具（豆包、DeepSeek、Kimi等）
5. 查看结果，如果不满意就追问

### 2. 注意事项
- ✅ 背景信息要具体（不要只说"学生基础一般"，要说"古文阅读能力中等，能理解基本字词"）
- ✅ 要求要明确（不要说"好一点"，要说"深入分析，有具体例子"）
- ✅ 可以多次追问（第一次不满意，就追问"能不能更具体一点"）
- ✅ AI的结果要检查（AI可能出错，你要把关）

### 3. 保存你的提示词
- 一旦写出好用的提示词，保存下来
- 下次遇到类似场景，改一改就能用
- 你的提示词库会越来越丰富

### 4. 推荐AI工具
- **豆包**（字节跳动）：对话流畅，免费
- **DeepSeek**：生成质量高，深度好
- **Kimi**：适合长文本处理
- **通义千问**（阿里）：稳定可靠

## 常见问题

**Q1：我用了三要素，但AI的回答还是不够好？**

可能原因：
- 背景信息还不够具体
- 要求不够明确
- AI的能力确实有限

解决办法：
- 多追问几次
- 换个AI试试
- 降低期待，把AI当助手

---

**Q2：AI生成的内容有错误怎么办？**

解决办法：
- AI是助手，不是替代品
- 你要检查AI的输出
- 不要直接用，要过一遍

---

**Q3：这个方法只适用于语文吗？**

不是！三要素法是通用的：
- 任何科目都能用
- 任何工作场景都能用
- 核心就是：说清楚你要什么

---

**记住：提示词不是技术，是清晰沟通。**  
**AI不是来取代你的，是来帮助你的。**
