## 生成任务说明

**课程主题：** 提示词三要素方法

**目标受众：** 中央文化和旅游管理干部学院学员

**核心内容：** 教会学员用"明确任务+丰富背景+细化要求"三要素方法，高效使用AI

**视觉主题：** 中国文旅元素（传统建筑、文化符号、景区场景）与现代科技融合

## 重要约束条件

### 中文显示优化约束

⚠️ **关键要求：避免复杂中文排版问题**

1. **文字布局约束：**
   - 图片上的中文尽量少，每张图不超过10个汉字
   - 优先用大标题（3-5个字）+ 图标/图形表达
   - 避免长段落文字覆盖在图片上
   - 重要信息用英文数字（1、2、3）而非中文序号（一、二、三）

2. **字体建议：**
   - 推荐使用黑体、思源黑体等粗体字
   - 避免使用宋体、楷体等细体字（在图片上不清晰）
   - 字号：标题至少36pt，正文至少20pt

3. **颜色对比：**
   - 文字必须有清晰背景色或描边
   - 深色背景用白字，浅色背景用黑字
   - 避免使用灰色、浅蓝等低对比度颜色

4. **信息层次：**
   - 复杂内容用「图标+数字」替代文字
   - 例如：用"1""2""3"而非"第一""第二""第三"
   - 例如：用"✓""×"而非"正确""错误"

5. **PPT特殊要求：**
   - 每页只有1个核心观点
   - 标题不超过15个字
   - 正文不超过50个字
   - 多用列表、图标、对比图

## 核心课程内容（供生成参考）

### 课程结构

**第一部分：为什么学提示词**
- 核心：同样的AI，效果差异来自提示词质量
- 类比：提示词=给AI的工作指令

**第二部分：三要素方法**
- 要素1：明确任务（告诉AI做什么）
- 要素2：丰富背景（告诉AI需要知道什么）
- 要素3：细化要求（告诉AI对输出的期望）
- 完整案例：文旅项目申报书

**第三部分：进阶技巧**
- 6个技巧：范例、分步、追问、角色、约束、风格
- 重点：风格仿写（去除AI味）

**第四部分：实战应用**
- 4步实战：选场景→三要素检查→写提示词→测试优化

**第五部分：总结**
- 核心：把需求说清楚
- 行动：立即实践

## 生成任务2：音频讲解（播客风格）

### 生成指令

```
请基于本文档内容，生成一段音频讲解脚本（播客对话风格）。

【音频形式】
- 形式：2人对话（老师+学员代表）
- 时长：20-30分钟
- 语气：专业但轻松，有互动感

【对话结构】
1. 开场（2分钟）
   - 老师：今天教大家用AI的核心方法
   - 学员：我们平时用AI效果不好，是哪里的问题？

2. 问题引入（3分钟）
   - 老师举例：同样问AI写工作总结，效果差这么多
   - 学员：原来是提示词的问题！

3. 三要素详解（15分钟）
   - 老师讲每个要素
   - 学员提问互动
   - 用文旅场景举例（项目申报、活动策划等）

4. 技巧介绍（5分钟）
   - 老师：掌握三要素后，还有进阶技巧
   - 重点讲风格仿写
   - 学员：这个太实用了！

5. 行动号召（2分钟）
   - 老师：今天就选个真实场景试试
   - 学员：好的，我先试试写工作总结

【中文口语化】
- 避免书面语，多用口语
- 举例要生动具体
- 适当加入"你看""比如说""其实"等口语词
- 学员的问题要代表真实困惑

【音频特殊要求】
- 语速适中，便于理解
- 重点内容重复2次
- 专业术语要解释
- 每个部分之间有过渡语

【内容来源】
参考本文档的"核心课程内容"部分。
```

## 通用约束（所有生成任务）

### 内容约束

1. **聚焦核心：** 只讲三要素方法，不展开所有进阶技巧
2. **案例真实：** 所有案例用文旅工作场景
3. **语言通俗：** 避免AI术语，多用工作中的真实表达
4. **行动导向：** 强调立即实践，不要只讲理论

### 文旅元素使用

1. **场景举例：**
   - 项目申报书
   - 活动策划方案
   - 工作总结报告
   - 宣传推广文案
   - 调研分析报告

2. **视觉元素：**
   - 传统建筑：园林、古镇、寺庙轮廓
   - 文化符号：书法、印章、祥云、水墨
   - 现代场景：景区、游客、导游、文创产品
   - 科技元素：几何图形、渐变色、数据可视化

3. **避免的元素：**
   - 过于复杂的写实照片
   - 纯古典风格（要现代感）
   - 与文旅无关的行业元素
   - 国外风景或建筑

### 技术优化

1. **分辨率：**
   - PPT：1920x1080
   - 信息图：1080x1350（适合手机）
   - 视频：1920x1080

2. **文件格式：**
   - PPT：.pptx或PDF
   - 音频：MP3
   - 视频：MP4
   - 信息图：PNG或JPG

3. **中文字体备选：**
   - 推荐：思源黑体、苹方、微软雅黑
   - 避免：宋体、楷体、艺术字体

## 使用建议

### 分开生成还是一起生成？

**建议：分开生成**

原因：
1. 每种格式对内容深度要求不同
2. PPT要简洁，音频要详细
3. 分开生成便于针对性优化

**生成顺序：**
1. 先生成PPT（结构最清晰）
2. 再生成信息图（提炼核心）
3. 然后生成音频（深度讲解）
4. 最后生成视频（综合展示）

### 如何上传到NotebookLM？

**方案1：一次性上传**
- 上传本文件 + 详细课程内容文档
- 在对话中指定生成哪一项
- 优点：上下文完整
- 缺点：可能内容太多

**方案2：分批上传**
- 每次生成一种格式
- 上传本文件 + 对应的生成指令部分
- 优点：聚焦具体任务
- 缺点：需要多次操作

**推荐：方案1（先试试，如果生成效果不好再用方案2）**

### 生成后如何优化？

1. **PPT：** 检查每页文字量，超过50字的拆成两页
2. **音频：** 试听后调整语速和停顿
3. **信息图：** 确保手机上也能看清文字
4. **视频：** 检查字幕与画面是否同步
