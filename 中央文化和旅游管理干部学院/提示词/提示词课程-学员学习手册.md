## 一、为什么要学提示词？

### 1.1 同样的AI，为什么效果差这么多？

**场景：** 你让AI帮你写一份工作总结

**随便问：**
> "帮我写一份本月工作总结"

**AI回答：**
- 内容空洞，都是套话
- 看不出你具体做了什么
- 和你的工作完全不搭

**细致问：**
> "我是XX市文旅局市场推广科科长，本月主要完成了3项工作：1）组织了"文化进景区"活动，覆盖8个景区，参与游客超2万人次；2）完成了全市文旅宣传资料改版，新增多语言版本；3）对接3家OTA平台，达成合作协议。请帮我写一份月度工作总结，要求：符合公文格式，字数800-1000字，既要体现数据成果，也要总结经验教训。"

**AI回答：**
- 内容具体，有数据有细节
- 结构专业，符合公文规范
- 可以直接使用或稍作修改

**差别在哪？** → 提示词的质量！

---

### 1.2 提示词的本质

**提示词 = 你给AI的工作指令**

- 指令越清楚，AI执行越准确
- 指令越模糊，AI只能靠猜
- AI不会主动问你细节，但你可以主动告诉它

**类比：**
- 就像你给下属安排工作：说清楚要做什么、为什么做、怎么做，下属才能做好
- 如果只说"去准备个方案"，下属只能瞎猜你想要什么

## 二、三要素方法详解

### 2.1 方法概述

**三要素 = 明确任务 + 丰富背景 + 细化要求**

这是一个通用框架，适用于任何场景：
- 写报告、写方案、写总结
- 生成文案、分析数据、整理资料
- ...你能想到的所有AI应用

---

### 2.2 要素1：明确任务

**核心：告诉AI你要它做什么**

#### ❌ 不好的任务描述
- "帮我处理这个"（什么叫处理？）
- "给我个建议"（什么方面的建议？）
- "帮我写点东西"（写什么？）

#### ✅ 好的任务描述
- "帮我写一份项目申报书"
- "帮我分析这份游客调研数据，找出3个核心问题"
- "帮我把这篇文章改成适合小红书发布的短文案"

#### 💡 检查标准
- [ ] AI能明确知道要"生成"还是"修改"还是"分析"？
- [ ] 任务的范围清楚吗？（不要太宽泛）
- [ ] 最终输出物是什么？（一篇文章？一份清单？一个方案？）

---

### 2.3 要素2：丰富背景

**核心：告诉AI它需要知道的信息**

AI只知道你告诉它的，不知道的它只能猜或者用通用模板。

#### 需要提供哪些背景？

**1. 你的身份/角色**
- 为什么重要？不同角色写出来的内容风格不同
- 例子："我是文旅局政策法规科干部" vs "我是景区运营经理"

**2. 工作场景/目标**
- 为什么重要？让AI理解上下文
- 例子："这是给市领导汇报的材料" vs "这是给游客看的宣传册"

**3. 关键信息**
- 为什么重要？AI不知道你的具体情况
- 例子：项目名称、时间地点、预算金额、涉及人数、政策依据等

**4. 目标受众**
- 为什么重要？同样的内容，对不同受众表达方式不同
- 例子："给专业人士看" vs "给普通游客看"

#### 实例对比

**缺背景：**
> "帮我写一份活动总结"

**有背景：**
> "我是XX景区市场部经理，上周举办了'非遗文化周'活动，为期7天，参与游客1.5万人次，举办了剪纸、泥塑等5项非遗体验活动，游客满意度92%，媒体报道8篇。请帮我写一份活动总结，目标是向景区管理层汇报，证明这个活动值得常态化开展。"

---

### 2.4 要素3：细化要求

**核心：告诉AI你对输出结果的具体期望**

#### 常见的要求维度

**1. 格式/结构要求**
- 文档格式：Word？PPT？纯文本？
- 文体类型：公文？报告？方案？新闻稿？
- 结构要求：要有哪些部分？
- 例子："符合公文格式，包含：背景、措施、成效、建议四部分"

**2. 长度/详细程度**
- 字数要求：多少字？
- 详细程度：简要概述还是详细阐述？
- 例子："控制在1500字以内" 或 "每个要点展开300字左右"

**3. 语气/风格**
- 正式程度：严肃正式 vs 轻松活泼
- 专业程度：专业术语 vs 通俗易懂
- 情感色彩：客观中立 vs 热情洋溢
- 例子："语言专业严谨，符合政府工作报告风格" 或 "语言活泼有趣，适合抖音短视频"

**4. 特殊约束**
- 必须突出的重点
- 必须避免的内容
- 必须符合的规范（如某个政策要求）
- 例子："重点突出文化传承与产业融合" 或 "避免使用专业术语，要让老百姓看得懂"

#### 实例对比

**缺要求：**
> 任务+背景都很清楚，但没说要求
> → AI会按自己的理解生成，可能不符合你的预期

**有要求：**
> "...（任务+背景）...
> 
> 要求：
> - 字数控制在2000-2500字
> - 结构：现状分析 → 存在问题 → 对策建议，三部分
> - 语言风格：专业但不枯燥，多用数据和实例支撑
> - 格式：符合调研报告规范
> - 重点突出：AI技术在文旅行业的落地可行性"

### 2.5 三要素组合示例

#### 完整示例1：项目申报材料

```
【任务】
请帮我撰写一份乡村文旅项目的省级示范村申报书。

【背景】
- 我的身份：XX县文旅局项目管理科科长
- 项目概况：XX村乡村旅游提升项目
- 村庄特点：有300年历史的古村落，保存完好的传统民居50栋
- 核心资源：省级非遗项目"剪纸艺术"，有3位国家级传承人
- 项目规划：打造"非遗体验+民宿集群"模式，建设20栋精品民宿+1个非遗传习所
- 投资预算：500万元（其中省级补贴申请300万）
- 申报目标：2025年度省级乡村旅游示范村

【要求】
- 符合《XX省乡村旅游示范村评定标准（2024版）》要求
- 包含以下部分：项目背景、资源优势、建设方案、预算分配、预期效益、保障措施
- 字数：5000-6000字
- 语言风格：专业严谨，符合政府申报材料规范
- 重点突出：文化传承与产业融合、带动村民就业增收
- 数据支撑：要有具体的游客预测、收益测算、就业岗位数据
```

#### 完整示例2：政策解读材料

```
【任务】
请帮我把《关于促进文化和旅游消费潜力的实施意见》这份政策文件，改写成一份面向景区企业的政策解读材料。

【背景】
- 我的身份：市文旅局政策法规科干部
- 任务目标：让全市100多家景区企业看懂政策，知道能享受什么优惠
- 政策核心内容：鼓励景区开发夜间旅游产品、支持文创产品开发、提供税收优惠和补贴政策
- 受众特点：景区企业负责人，不是政策专家，需要通俗易懂
- 发布渠道：会在文旅局官网和微信公众号发布

【要求】
- 结构：政策要点 → 支持措施 → 申报流程 → 案例参考
- 字数：3000字左右
- 语言风格：通俗易懂，少用政策术语，多用大白话
- 格式：每个政策点用"Q&A"形式呈现（问：企业最关心的问题，答：政策解释+申报方式）
- 必须包含：具体的补贴金额、申报时间、联系方式
- 配图建议：提供2-3个典型案例，说明哪些企业已经享受到政策红利
```

## 三、进阶技巧

掌握三要素后，这些技巧能让你的提示词更强大：

---

### 3.1 技巧1：提供范例

**原理：** 与其描述你要什么，不如直接给AI看一个好例子

**适用场景：**
- 需要特定格式（如公文、表格）
- 需要特定风格（如某人的写作风格）
- 需要模仿某个成功案例

**示例：**

```
我需要写一份景区宣传文案。下面是我之前写的一篇效果很好的文案：

「示例文案」
走进XX古镇，就像翻开一本泛黄的线装书。青石板路承载着300年的故事，
老手艺人的剪刀声清脆悠扬。这里没有商业化的喧嚣，只有烟火气的温柔。
清晨，看老人打太极；午后，听评弹悠扬；夜晚，赏灯影摇曳。XX古镇，
一个让时光慢下来的地方。

请参考这个风格和结构，为YY水乡古镇写一篇宣传文案。YY水乡特点是...（补充背景信息）
```

**效果：** AI会模仿范例的风格、句式、节奏，生成风格一致的内容

---

### 3.2 技巧2：分步骤提问

**原理：** 复杂任务拆成多个小步骤，AI每次只做一件事，准确度更高

**适用场景：**
- 任务很复杂，一次性说不清
- 需要根据AI的回答再决定下一步
- 想要有互动感的创作过程

**示例：**

```
步骤1：
我要写一份文旅行业调研报告，主题是"AI在景区运营中的应用现状"。
请先帮我设计一个调研提纲，包括：调研目的、调研对象、调研问题清单。

（AI回答后）

步骤2：
很好！基于这个提纲，我已经完成了对10家景区的调研。调研发现主要有三个问题...
请帮我把这些问题整理成一份"现状分析"部分，要求：逻辑清晰，数据支撑，字数1500字左右。

（AI回答后）

步骤3：
现状分析写得不错。接下来请基于这些问题，提出5条可行的改进建议...
```

**效果：** 每步都能调整方向，最终输出更符合预期

---

### 3.3 技巧3：追问深化

**原理：** AI的第一次回答不一定完美，通过追问可以不断优化

**适用场景：**
- AI的回答方向对，但不够深入
- AI的回答有些地方需要调整
- 想要探索更多可能性

**常用追问方式：**

**1. 要求展开**
> "第3点'提升服务质量'这一条，请展开详细阐述，包括具体措施和预期效果"

**2. 要求调整风格**
> "这个方案写得太正式了，请改成轻松活泼的风格，适合在微信公众号发布"

**3. 要求增加内容**
> "请在这个方案中增加'成本预算'和'风险控制'两个部分"

**4. 要求提供选项**
> "这个标题不够吸引人，请再给我5个不同风格的标题选项"

**5. 要求修正错误**
> "你提到的'XX政策'理解有误，实际上是...，请基于正确理解重新写这一段"

---

### 3.4 技巧4：风格仿写

**原理：** 让AI学习特定的写作风格，去除"AI味"

**适用场景：**
- AI生成的内容太套路化
- 需要符合个人或组织的语言风格
- 需要与已有内容保持一致性

**操作步骤：**

```
步骤1：准备风格样本
（你自己写的文章，或者某位作家、专家的文章片段）

步骤2：让AI学习风格
请分析下面这段文字的写作风格特点：

「样本文字」
文旅AI融合，说到底就三件事：降本、增效、提质。降本，是把内容生产成本降下来
——过去一个景区导览词外包要5万，AI生成只要500；增效，是把响应速度提上去
——游客咨询不用等，AI顾问24小时在线；提质，是把服务质量稳定住——人工服务
参差不齐，AI可以保证基准线。

步骤3：要求模仿
现在请用上述风格，写一段关于"AI在文旅营销中的应用"的内容，要求：
- 保持"说到底就...""是把...""——"的句式特点
- 保持务实、直白、有数据对比的风格
- 字数300字左右
```

**效果：** 生成的内容更有个人特色，不那么"AI味"

---

### 3.5 技巧5：角色扮演

**原理：** 让AI扮演特定角色，输出更专业的内容

**示例：**

```
请你扮演一位有20年经验的文旅项目评审专家。现在有一份乡村旅游项目申报书，
请从专家角度对其进行评审，指出：
1. 哪些地方写得好？
2. 哪些地方有问题？
3. 如果是你评审，你会重点关注哪些指标？

（附上申报书内容）
```

**效果：** AI会从专业角色出发，给出更有深度的分析

---

### 3.6 技巧6：设置约束

**原理：** 明确告诉AI"不要什么"，避免出现不想要的内容

**示例：**

```
帮我写一份文旅发展规划方案，要求：

✅ 要包含：现状分析、发展目标、重点项目、保障措施
✅ 要突出：可行性、创新性、可持续性

❌ 不要：空洞的口号和套话
❌ 不要：没有数据支撑的判断
❌ 不要：照搬其他地区的做法
❌ 避免：过于理想化的目标设定

...（其他背景信息）...
```

**效果：** AI会更注意避开你不想要的内容

## 四、文旅场景案例库

### 4.1 政策与规划类

#### 案例1：政策文件解读

**场景描述：** 上级下发了一份政策文件，需要解读给基层单位或企业

**提示词模板：**
```
【任务】
请帮我把《政策文件名称》改写成一份通俗易懂的政策解读材料。

【背景】
- 我的身份：[你的岗位]
- 目标受众：[基层单位/景区企业/从业人员]
- 政策核心内容：[简要概述政策要点]
- 发布目的：让受众看懂政策，知道如何申报/享受优惠

【要求】
- 结构：政策背景 → 核心要点 → 申报指南 → 常见问题
- 语言：通俗易懂，少用术语
- 格式：Q&A形式，每个问题配答案
- 必须包含：申报条件、申报时间、申报流程、联系方式
```

---

#### 案例2：文旅发展规划

**场景描述：** 需要编写本地区的文旅发展规划

**提示词模板：**
```
【任务】
请帮我编写《XX地区"十四五"文化和旅游发展规划（2024-2028）》。

【背景】
- 我的身份：[区县文旅局规划科负责人]
- 地区概况：[人口、面积、经济水平、文旅资源]
- 现状：[游客量、旅游收入、主要景区、存在问题]
- 上级要求：[符合市级、省级规划，突出某方面特色]

【要求】
- 包含：现状分析、SWOT分析、发展目标、重点任务、保障措施
- 字数：15000字左右
- 目标量化：游客量、旅游收入等要有具体数字
- 项目清单：列出5-8个重点项目，包括投资额、建设周期
- 风格：专业严谨，符合政府规划文本格式
```

### 4.2 项目类

#### 案例3：项目申报材料

**场景描述：** 申报省级/国家级文旅项目资金支持

**提示词模板：**
```
【任务】
请帮我撰写[项目名称]的[省级/国家级][项目类型]申报书。

【背景】
- 项目概况：[项目名称、位置、类型]
- 资源特点：[文化资源/旅游资源/特色优势]
- 建设内容：[要做什么，分几期，每期做什么]
- 投资预算：[总投资，申请补助金额]
- 预期效益：[经济效益、社会效益、文化效益]

【要求】
- 符合[具体评选标准/申报指南]要求
- 包含：项目背景、必要性分析、建设方案、投资估算、效益分析、保障措施
- 字数：[根据申报要求]
- 重点突出：[创新性/示范性/带动性/可持续性]
- 数据支撑：要有游客预测、收益测算、就业带动等量化指标
```

---

#### 案例4：项目评估报告

**场景描述：** 对已完成项目进行效果评估

**提示词模板：**
```
【任务】
请帮我撰写[项目名称]的项目评估报告。

【背景】
- 项目基本情况：[建设时间、投资额、主要内容]
- 目标完成情况：[对比申报时的目标，实际完成了哪些]
- 数据情况：[游客量、收入、满意度等关键数据]
- 存在问题：[实施过程中遇到的问题]

【要求】
- 结构：项目概况 → 实施情况 → 目标完成度分析 → 效益评估 → 经验总结 → 改进建议
- 字数：5000-6000字
- 评估维度：经济效益、社会效益、环境效益、管理效益
- 风格：客观公正，有数据有分析
```

## 五、常见问题解答

### 5.1 关于提示词编写

**Q1：提示词是不是越长越好？**

A：不是。好的提示词是"信息密度高"，不是"字数多"。

- ✅ 好：提供关键信息，删除废话
- ❌ 不好：重复啰嗦，信息量少

**Q2：三要素的顺序必须固定吗？**

A：不必须，但推荐"任务→背景→要求"的顺序。

- 这个顺序符合人的思维习惯
- 先说做什么，再说为什么，最后说怎么做
- 但如果你习惯其他顺序也可以

**Q3：每次都要写这么详细吗？**

A：看情况。

- 简单任务：只说清任务就够了
- 复杂任务：三要素都要齐全
- 判断标准：AI是否有足够信息完成任务

---

### 5.2 关于AI回答

**Q4：AI的回答不满意怎么办？**

A：三种策略：

1. **追问优化**：告诉AI哪里不满意，让它改进
2. **调整提示词**：重新写提示词，补充信息或调整要求
3. **换个角度**：换一个问法，可能效果更好

**Q5：AI生成的内容能直接用吗？**

A：建议人工审核和修改。

- AI可能出现事实错误（尤其是数据、政策）
- AI可能理解偏差
- AI生成的内容可能需要根据实际情况调整

**使用建议**：把AI当助手，不当替代品
- AI给你80%，你完善20%
- AI提供思路，你决策和把关

**Q6：为什么同样的提示词，每次AI回答不一样？**

A：这是AI的特点。

- AI每次生成都有一定随机性
- 不是bug，是设计如此
- 好处：可以多生成几次，选最好的

---

### 5.3 关于实际应用

**Q7：我的工作场景很特殊，案例库里没有怎么办？**

A：用三要素方法自己设计。

步骤：
1. 明确任务：我要AI做什么？
2. 丰富背景：AI需要知道什么？
3. 细化要求：我对输出有什么具体期望？

**Q8：使用AI会不会泄露工作机密？**

A：注意信息安全。

- 不要把涉密文件直接粘贴给AI
- 可以脱敏处理（隐去具体地名、人名、数字）
- 敏感内容自己写，让AI优化格式和语言

**Q9：领导说AI生成的太"AI味"，怎么办？**

A：使用"风格仿写"技巧。

- 找一篇你们认可的文章
- 让AI学习这个风格
- 再生成内容就会更像人写的

**Q10：我不太会用电脑，能学会吗？**

A：能！

- 提示词就是跟AI聊天，会打字就行
- 不需要编程、不需要技术背景
- 就像给下属布置任务，说清楚就行
