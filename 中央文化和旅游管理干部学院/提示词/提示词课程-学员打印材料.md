
# 从弱到强的提示词演示

## 案例：编写文旅行业研究报告

---

## ❌ 弱提示词示例

**提示词：**
```
帮我把这份文旅AI融合报告写成一本书的大纲
```

**AI生成效果：**
- 章节标题空洞（第一章：概述，第二章：现状...）
- 缺乏针对性（看不出文旅特色）
- 结构通用化（任何行业都能套用）
- 没有实施指导（理论多，实操少）

---

## ✅ 强提示词示例（加入三要素）

**提示词：**
```
【任务】
我要把这份"文旅AI融合应用分析报告"扩展成一本面向文旅行业的
实战指南，请帮我设计详细的章节大纲。

【背景】
- 我的身份：文旅管理部门干部
- 目标读者：文旅行业从业者（景区管理者、文旅局干部、创业者）
- 报告核心内容：文旅行业痛点分析 + AI解决方案（智能导览、
  营销内容、游客服务、文化传承、数据决策、多语言服务六大方向）
- 写作目的：让读者看完能理解AI价值，知道从哪入手，
  能找到适合自己的应用方向

【要求】
- 全书结构：理论框架 + 实战案例 + 实施路线图
- 每章包含：痛点分析、解决方案、技术实现、落地案例、行动清单
- 控制在10-15万字，适合6-8个月完成
- 语言风格：专业但不枯燥，有数据支撑但通俗易懂，
  多用文旅行业真实案例
- 特别要求：第一章要有冲击力，让读者立即看到AI的价值
```

**AI生成效果：**
- 章节针对性强（紧扣文旅场景）
- 结构实战化（从认知到落地）
- 内容可操作（每章都有行动指南）
- 读者定位清晰（知道为谁写）

---

## 🎯 三要素方法总结

✅ **明确任务** - AI要做什么？（写/改/分析/扩展...）

✅ **丰富背景** - AI需要知道什么？（身份/场景/目的...）

✅ **细化要求** - 对输出有什么具体要求？（格式/长度/风格...）

**关键：** 这个方法适用于所有场景（报告/方案/总结/文案...）
