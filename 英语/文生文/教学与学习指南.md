# 文生文课程 - 学习指南（英语科）

> **课程目标：** 掌握现成的提示词，改几个关键词就能用  
> **课程性质：** 实操课，不讲理论，重点是效果和实践

---

## 课程内容

本课程提供4个真实场景的AI提示词，帮助你快速解决英语教学中的常见问题：

| 场景 | 解决什么问题 |
|------|-------------|
| 场景1：生成80分范文 | 快速生成符合要求的范文 |
| 场景2：对比找差距 | 用AI分析差距，教学生写好作文 |
| 场景3：苏格拉底式对话 | 和AI讨论，理清教学思路 |
| 场景4：分层教学 | 针对不同水平学生给建议 |

## 场景1：生成80分范文

### 使用场景
- 你需要一篇好范文用于课堂讲解
- 你手头范文不够新/不够好
- 你想给学生提供不同类型的范文参考

### 完整提示词

#### 变体1：议论文（最常用）

```
【任务】
请写一篇高考英语议论文范文。

【背景】
- 题目：Should students be required to participate in community service?
- 目标分数：85分以上（满分120分）
- 学生年级：高三
- 作文要求：120-150词，观点明确，论证充分，正反对比
- 使用目的：课堂讲解示范，展示高分作文特点

【要求】
1. 结构要求：
   - 引入段：引出话题，表明立场
   - 正方论证：2个论据，有深度
   - 反方论证：1个论据，体现思辨
   - 结论段：重申立场，升华观点

2. 语言要求：
   - 高级词汇：至少使用8-10个高级词汇（标注出来）
   - 复杂句式：定语从句、状语从句、非谓语动词各至少1处
   - 连接词：使用恰当的过渡词（admittedly, furthermore, however等）

3. 论证要求：
   - 不要空洞说理，要有具体分析
   - 论据要有说服力
   - 体现批判性思维

4. 输出格式：
   - 范文正文
   - 高级词汇清单（词汇-中文-为什么好）
   - 复杂句式标注（句子-类型-效果）
   - 评分预估及理由
```

**使用建议：**
- 根据需要修改题目
- 目标分数可以调整（80分/85分/90分）
- 如果是高一高二，降低词汇难度要求

---

#### 变体2：应用文（申请信/建议信等）

```
【任务】
请写一篇高考英语应用文范文（申请信）。

【背景】
- 题目：申请参加学校英语戏剧社
- 目标分数：80分以上（满分120分）
- 学生年级：高二
- 作文要求：100-120词，格式规范，表达得体
- 使用目的：教学生写申请信的标准格式和表达

【要求】
1. 格式要求：
   - 称呼、正文、结尾、署名规范
   - 段落结构清晰（自我介绍-申请理由-期待回复）

2. 内容要求：
   - 自我介绍（身份、特长）
   - 申请理由（2-3条，有说服力）
   - 表达意愿和期待

3. 语言要求：
   - 语气得体、礼貌
   - 使用申请信常用句式
   - 避免口语化表达

4. 输出格式：
   - 范文正文
   - 格式说明（每个部分的作用）
   - 常用句式清单
   - 评分预估及理由
```

---

#### 变体3：读后续写/概要写作

```
【任务】
请写一篇高考英语读后续写范文。

【背景】
- 原文主题：一个关于诚实的故事（学生在考试中面临作弊诱惑）
- 目标分数：80分以上（满分25分，读后续写部分）
- 学生年级：高三
- 续写要求：150词左右，与原文衔接自然，情节合理，语言连贯
- 使用目的：教学生如何续写，把握情节和语言风格

【要求】
1. 情节要求：
   - 与原文情节自然衔接
   - 情节发展合理，有转折
   - 体现主题（诚实的重要性）

2. 语言要求：
   - 与原文风格一致
   - 使用细节描写（动作、心理、环境）
   - 对话要自然

3. 结构要求：
   - 两段续写，每段75词左右
   - 第一段：矛盾冲突
   - 第二段：解决问题，升华主题

4. 输出格式：
   - 续写正文（两段）
   - 衔接点分析（如何与原文衔接）
   - 语言亮点标注
   - 评分预估及理由

（需提供原文内容）
```

---

### 注意事项

1. **AI生成的范文需要人工审核**
   - 检查有无语法错误
   - 检查论证是否合理
   - 检查是否符合中国学生实际

2. **不同AI工具效果可能不同**
   - DeepSeek效果相对稳定
   - 如果一次效果不好，多生成几次选最好的

3. **根据学生水平调整**
   - 高一高二：降低词汇和句式要求
   - 基础班：可以要求生成"75分"而不是"85分"

## 场景2：对比找差距（60分 vs 85分）

### 使用场景
- 你想教学生"好作文和普通作文的差距在哪"
- 你想给学生讲解如何从60分提升到80分以上
- 你想让学生看到真实的好作文和普通作文的具体差距

### 完整提示词

```
【任务】
请对比分析这两篇学生作文，说明它们的差距在哪里。

【背景】
- 作文A：60-70分水平（基础薄弱学生）
- 作文B：85分水平（优秀学生）
- 对比目的：用于课堂讲解，让学生看到好作文和普通作文的具体差距
- 题目：Should students be allowed to use smartphones in school?

【要求】
1. 逐项对比（用表格）：
   - 语法：作文A的典型错误 vs 作文B的正确表达
   - 词汇：作文A的简单词 vs 作文B的高级词（举例5处）
   - 句式：作文A的单一句式 vs 作文B的复杂句式（举例3处）
   - 论证：作文A的浅显论证 vs 作文B的深入分析（具体说明）
   - 整体：给人的感觉有什么不同（评卷老师视角）

2. 提升路径：
   - 如果作文A想达到作文B的水平，优先改什么（前3条）
   - 每条预计能提升多少分
   - 为什么这3条最重要

3. 教学建议：
   - 如果课堂上讲这个对比，重点讲哪3点最有说服力
   - 每个点用这两篇作文的哪个具体例子
   - 学生最容易理解的讲解顺序

【作文A（60-70分）】
（粘贴示例作文-60分-语法崩溃型.md 或 68分-表达幼稚型.md）

【作文B（85分）】
（粘贴示例作文-85分-细节待优化型.md）
```

---

### 推荐的作文组合

**组合1（推荐）：60分 vs 85分**
- 差距最明显，对比效果最好
- 适合大部分学生看
- 文件：示例作文-60分.md + 示例作文-85分.md

**组合2：68分 vs 85分**
- 适合讲词汇和表达升级
- 语法基本对，重点在提升
- 文件：示例作文-68分.md + 示例作文-85分.md

**组合3：75分 vs 85分**
- 差距较小，适合冲刺班
- 重点讲论证深度和细节
- 文件：示例作文-75分.md + 示例作文-85分.md

**推荐：用组合1（60分 vs 85分），对比最明显**

---

### 注意事项

1. **选择合适的作文组合**
   - 60分 vs 85分：差距最明显，对比最震撼（推荐）
   - 68分 vs 85分：适合讲词汇和表达升级
   - 75分 vs 85分：适合冲刺班，讲细节优化

2. **两篇作文要同一题目**
   - 否则对比没意义
   - 我们的5篇示例作文都是同一题目，可以任意组合

3. **关注具体例子**
   - AI会给出对比表格
   - 重点看具体例子：60分怎么写，85分怎么写
   - 这些例子最有说服力

4. **提升路径要可操作**
   - AI给出的优先级很重要
   - 如果在课堂上讲，重点讲前3个优先级
   - 告诉学生"改这3条，就能提升20分"

## 场景3：苏格拉底式对话

### 使用场景
- 你有教学困惑，但不确定具体怎么问
- 你想和AI讨论，理清思路
- 你想探索不同的教学方案

### 完整提示词

```
你是一位资深英语教研专家，擅长用苏格拉底式提问帮助老师理清教学思路。

当英语老师向你描述教学困惑或需求时，请通过3-5轮提问，
帮助老师：
1. 明确问题本质
2. 理清现状和目标
3. 找到可行的解决方案

【提问原则】
- 每次只问1-2个具体问题，不要一次问太多
- 基于老师的回答，逐步深入
- 问题要具体、接地气，符合英语教学实际
- 避免抽象的教育理论术语
- 最后给出可操作的建议（3-5条，有优先级）

【对话流程示例】
老师："我想提升学生的英语作文水平"
你："很好的目标。请问学生目前的主要问题是什么？
是语法错误多，还是词汇贫乏，还是论证不够深入？"

老师回答后 → 你继续提问，逐步明确：
- 学生现状（分数段、典型问题）
- 老师目标（提升多少分、时间要求）
- 可用资源（课时、学生配合度）
- 教学偏好（喜欢什么方式）

最后给出：
- 3-5条具体建议
- 每条建议包括：做什么、怎么做、预期效果
- 建议要有优先级

---

现在，请帮我理清这个问题：

【我的困惑/需求】
________________（老师填写）

请开始提问。
```

---

### 使用示例

#### 示例1：提升作文水平

**老师输入：**
> "我想让学生把英语作文从60-70分提升到80分以上，但不知道从哪里入手。"

**AI第1轮提问：**
> "明白了。首先想了解一下学生的具体情况：
> 1. 学生目前作文60-70分，主要失分在哪里？是语法错误多，还是词汇简单，还是论证不够？
> 2. 你希望多久达到这个目标？一个月、一学期，还是一学年？"

**老师回答：**
> "主要是词汇太简单，论证也比较浅显。时间大概一个学期。"

**AI第2轮提问：**
> "好的。针对词汇和论证这两个问题：
> 1. 你每周有多少课时可以专门用于作文教学？
> 2. 学生对作文练习的配合度如何？愿意课后练习吗？"

**...(继续3-5轮)**

**AI最后给建议：**
> "基于我们的讨论，我建议按以下优先级推进：
> 
> 【优先级1】词汇升级训练（重要且见效快）
> - 做什么：每周给学生10个高级替换词（如very→extremely, good→outstanding）
> - 怎么做：让学生用这些词改写自己的作文句子
> - 预期效果：2-3周后，学生词汇丰富度明显提升，预计作文提升5-8分
> 
> 【优先级2】论证深度训练（重要但需要时间）
> - 做什么：教学生用"举例-分析-总结"三步展开论证
> - 怎么做：每次作文给一个论证模板，让学生套用练习
> - 预期效果：4-6周后，学生论证有深度，预计再提升5-8分
> 
> 【优先级3】范文仿写（巩固提升）
> - 做什么：每两周分析一篇85分以上范文
> - 怎么做：标注范文亮点，让学生模仿写一篇
> - 预期效果：帮助学生建立"好作文"的概念
> 
> 建议：前4周重点抓优先级1，见效后再加入优先级2。"

---

### 注意事项

1. **要真诚回答AI的问题**
   - AI的提问是为了理解你的情况
   - 回答越具体，建议越有针对性

2. **可以随时追问**
   - 如果建议不够清楚，继续问
   - "第1条能详细说说怎么做吗？"
   - "有没有更简单的办法？"

3. **建议要结合实际**
   - AI的建议是通用的
   - 要根据自己学生的实际情况调整

## 场景4：分层教学

### 使用场景
- 班上学生水平差异大
- 你的精力有限，无法一对一指导
- 你想针对不同水平给不同建议

### 完整提示词

```
【任务】
请为我班上不同水平的学生设计分层作文教学方案。

【背景】
- 年级：高二
- 班级特点：学生英语水平差异较大
- 教学目标：帮助每个层次的学生都有提升
- 时间安排：一个学期（约16周）

【学生分层情况】
A类学生（约30%）：
- 目前分数：50-65分（满分120分）
- 主要问题：语法错误多，词汇量小，句子不完整
- 目标分数：70-75分

B类学生（约50%）：
- 目前分数：70-80分（满分120分）
- 主要问题：词汇简单，句式单一，论证浅显
- 目标分数：85-90分

C类学生（约20%）：
- 目前分数：85-95分（满分120分）
- 主要问题：细节不够精致，缺少"惊艳"的表达
- 目标分数：100分以上

【要求】
请分别为三类学生设计方案，包括：

1. 问题诊断：
   - 每类学生的核心问题（3条）
   - 这些问题为什么导致失分

2. 提升策略：
   - 重点训练什么（按优先级排序）
   - 用什么方法训练
   - 大概需要多长时间见效

3. 具体行动：
   - 每周课堂上做什么（10分钟内完成）
   - 每周课后做什么（20分钟内完成）
   - 需要老师批改什么

4. 评估标准：
   - 怎么判断学生有进步
   - 多久评估一次
   - 达到什么标准可以升到下一层

5. 注意事项：
   - 三类学生的训练能否在同一课堂进行
   - 如果不能，怎么安排
   - 老师精力分配建议
```

---

### 注意事项

1. **分层标准要灵活**
   - 不一定是3层，可以是2层或4层
   - 根据班级实际情况调整

2. **避免标签化**
   - 分层是为了教学，不是给学生贴标签
   - 学生可以流动（进步了就升层）

3. **老师精力有限**
   - 不可能三类学生都花同样精力
   - 重点关注中间层（人数最多，提升空间大）
   - A类抓基础，C类可以自主学习

## 常见问题

**问题1："AI生成的范文有语法错误怎么办？"**
> "正常的，AI也会犯错。你看一遍，有错误就追问AI：'第3句语法有误，请修正'。"

**问题2："我的学生作文和示例作文不一样，能用吗？"**
> "当然能用。行动卡上的提示词是通用的，你把自己学生的作文粘贴进去就行。"

**问题3："生成的范文能直接给学生吗？"**
> "建议你先看一遍，确认没问题再给学生。AI是助手，你是把关人。"

**问题4："苏格拉底式对话要聊多久？"**
> "一般3-5轮就能得到建议。如果AI问的问题你答不上来，说明这个问题值得你思考。"
