## 正面：完整案例 - 作文批改

### 弱提示词 vs 强提示词对比

#### ❌ 弱提示词（效果不够精准）

```
帮我批改这篇英语作文。

（作文内容）
```

**AI回答：**
- 笼统评价："这篇作文整体不错..."
- 泛泛而谈："注意语法错误"、"词汇可以更丰富"
- 没有具体建议
- 老师/学生看完：还是不知道怎么改

---

#### ✅ 强提示词（效果精准可用）

```
【任务】
请批改这篇高中英语作文，给出详细反馈和修改建议。

【背景】
- 学生情况：高二学生，英语中等水平，平时考试80-85分（满分120），
  目标突破到95分以上
- 作文类型：议论文（正反对比型），题目是"Should students be 
  allowed to use smartphones in school?"
- 写作要求：120-150词，观点明确，论证充分，使用适当的连接词
- 批改目的：这是学生的第一稿，需要具体的修改方向，帮助学生理解
  "好作文和普通作文的差距在哪"

【要求】
请从以下4个维度批改：

1. 语法错误诊断（逐句检查）
   - 列出所有语法错误：主谓一致、时态、代词、冠词、介词等
   - 用表格呈现：原句 | 错误类型 | 修正后
   
2. 词汇升级建议（至少找出8-10处）
   - 找出过于简单/重复的词汇
   - 给出高级替换词（适合高二学生水平，不要太难）
   - 说明为什么这个词更好
   
3. 句式优化建议
   - 指出哪些句子可以合并成复杂句（定语从句、状语从句、非谓语）
   - 给出2-3个具体的改写示例
   
4. 论证深度提升
   - 指出论证太浅的地方
   - 建议如何展开（举例、对比、因果分析）

5. 分数预估与改进优先级
   - 当前预估分数（/120）及理由
   - 按重要性排序：如果只改3处，应该改哪3处？
   - 改进后预估能达到多少分

（作文内容）
```

**AI回答：**
- ✅ 语法错误用表格，一目了然，具体到每个句子
- ✅ 词汇升级很具体：very popular → increasingly prevalent
- ✅ 句式改写有示例：两个简单句 → 一个复杂句
- ✅ 论证建议可操作：不只说"太浅"，还告诉怎么展开
- ✅ 给出分数预估和优先级：先改什么、后改什么

---

### 核心差异

| 对比项 | 弱提示词 | 强提示词 |
|-------|---------|---------|
| **学生信息** | 没有 | 高二、80-85分、目标95分 |
| **批改维度** | 笼统 | 4个明确维度 |
| **详细程度** | 泛泛而谈 | 逐句检查、8-10处词汇 |
| **输出格式** | 段落 | 表格、对比、示例 |
| **可操作性** | 不知道改什么 | 有优先级，先改什么 |
| **用时** | 2分钟看完，还是不懂 | 10分钟研究，完全能改 |

### 三要素方法

**✅ 明确任务（AI要做什么？）**
- 批改作文？出阅读题？生成范文？设计教案？
- 输出什么内容？
- 任务范围清楚吗？

**✅ 丰富背景（AI需要知道什么？）**
- 学生情况（年级、水平、目标）
- 教学目标（巩固基础？冲刺高分？）
- 具体要求（作文类型、考试标准）
- 使用目的（第一稿？考前精批？）

**✅ 细化要求（对输出有什么具体期望？）**
- 批改维度（语法？词汇？句式？论证？）
- 详细程度（简要？详细？逐句？）
- 输出格式（表格？列表？对比？）
- 优先级（最重要的3个问题是什么？）

---

**记住：好的提示词 = 把你的需求说清楚**

就像给学生布置作业，说得越清楚，做得越好。

## 背面：行动卡（实战练习用）

### 使用说明

1. **选择场景**：从下面5个常见场景选一个，或用你自己的真实场景
2. **填写模板**：在模板的空白处填写你的具体信息
3. **检查三要素**：用检查清单确认是否完整
4. **问AI测试**：复制粘贴到AI界面，看效果
5. **优化迭代**：不满意就调整，再试

### 三要素检查清单

在提交给AI之前，用这个清单检查你的提示词：

#### ✅ 明确任务（AI要做什么？）
- [ ] 任务清楚吗？（批改/出题/生成/设计）
- [ ] 输出内容明确吗？
- [ ] 任务范围清楚吗？

#### ✅ 丰富背景（AI需要知道什么？）
- [ ] 学生情况清楚吗？（年级、水平、目标）
- [ ] 教学目标明确吗？
- [ ] 具体要求说明了吗？（类型、标准、用途）
- [ ] 使用场景清楚吗？

#### ✅ 细化要求（对输出有什么期望？）
- [ ] 维度/结构要求清楚吗？
- [ ] 详细程度说明了吗？
- [ ] 输出格式指定了吗？
- [ ] 有优先级或特殊要求吗？

**如果打勾都完成 → 可以提交给AI了！**

**如果有遗漏 → 补充后再提交**

### 实战提示

**💡 第一次使用？**
- 先用"作文批改"场景，最容易上手
- 照着模板填，不要空着提交

**💡 AI回答不满意？**
- 追问："请详细展开第3点"
- 调整："请改用表格形式呈现"
- 补充："请再给5个例句"

**💡 不确定怎么写？**
- 看正面的完整案例
- 问讲师或旁边老师
- 先写一个简单版，再逐步完善
